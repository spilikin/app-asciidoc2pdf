= ascidoc2pdf: Konvertiert Asciidoc nach PDF in gematik Template
:source-highlighter: rouge

== Ausführen über Docker Compose
[source,bash]
----
docker compose build
docker compose run --rm asciidoc2pdf \-D build README.adoc
----
